# Shell variables and expression

## Usage
* ./configure

      ==== option show =====
      options number: 0

      ==== :- expression test ====
      ./configure
      default
      default
      
* ./configure  Bepartofyou

      ==== option show =====
      Bepartofyou
      options number: 1

      ==== :- expression test ====
      ./configure
      Bepartofyou
      default
      
* ./configure  Bepartofyou one two three four five

      ==== option show =====
      Bepartofyou
      one
      two
      three
      four
      five
      options number: 6

      ==== :- expression test ====
      ./configure
      Bepartofyou
